**English:**

This is my portfolio, where I can share some code and present it on GitPages. I plan to create various codes, even with limited time. Initially, you can explore code examples related to computer vision, including models created using YOLO, TensorFlow, and PyTorch, as well as exports to TensorFlow.js.

**Project 1: Object Detection Library**

Installation: `npm i object-detection-from-yolo-to-tensorflow`

Repository: [GitHub Link](https://github.com/VictorBarbosa/NpmObjectDetection)

**Project 2: Mask Detector**

In this example, I've developed an object detection model using YOLO to detect whether a person is wearing a mask or not.

Repository: [GitHub Link](https://github.com/VictorBarbosa/mask-detector)

Live Test: [Git Pages](https://victorbarbosa.github.io/mask-detector/)

**Project 3: Obscene Gesture Detector**

In this example, I've developed an object detection model using YOLO to detect if a person is making an obscene gesture (🖕). If positive, a blur will be applied over the image.

Repository: [GitHub Link](https://github.com/VictorBarbosa/ObsceneGestureDetector)

Live Test: [Git Pages](https://victorbarbosa.github.io/ObsceneGestureDetector/)

**Project 4: Finger Tracking and Canvas Drawing**

In this example, I've developed an object detection model using YOLO to track the movement of the index finger and draw on a canvas, showing the path the finger has taken.

Repository: [GitHub Link](https://github.com/VictorBarbosa/draw-on-canvas-with-finger)

Live Test: [Git Pages](https://victorbarbosa.github.io/draw-on-canvas-with-finger/)

Note: This may not work on smartphones and/or tablets; it has been optimized for browsers only.

**Project 5: AI Playing Dino Chrome Game**

In this example, I have created an AI that plays the Google Chrome's Dinosaur game.

#### Key Points

- I didn't build the game from scratch; I incorporated the source code that I found on the internet [Here](https://codepen.io/MysticReborn/pen/rygqao).
- I had to make some modifications since I needed a callback whenever obstacles moved on the screen, to determine distance and speed.
- Although it worked, there are random events in the game that can cause the AI to lose the game, for example, the game's speed.

Repository: [GitHub Link](https://github.com/VictorBarbosa/ai-playing-dino-chrome/)

Live Test:[Git Pages](https://victorbarbosa.github.io/ai-playing-dino-chrome/)

Note: This may not work on smartphones and/or tablets; it has been optimized for browsers only.
